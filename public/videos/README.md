# Hero Video

Colocá el video como:
  public/videos/hero.mp4

El video debe ser:
- Formato: MP4 (H.264)
- Resolución: 1920x1080 mínimo
- Sin audio (o el audio se ignorará — el video es muted)
- El video inicia completamente negro/oscuro ya que el site theme es negro
- GSAP anima la opacidad gradualmente cuando carga la página

Si no tenés el video todavía, el hero muestra igual (fondo negro con texto animado).
